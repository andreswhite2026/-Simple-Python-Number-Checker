# Narration Script

Hello, my name is Andrés Blanco.

Today I am explaining my Python program.

This project is a simple Python program.
It is a number checker.

The program checks if a number is positive, negative, or zero.

First, the program asks the user to enter a number.

The variable number is storing the value from the user.

Now the system is checking the number.

The program compares the number with zero.

If the number is greater than zero, the program prints that the number is positive.

If the number is less than zero, the program prints that the number is negative.

If the number is equal to zero, the program prints that the number is zero.

The program shows the result on the screen.

The program is currently processing the user input and showing the result.
